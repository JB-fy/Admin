依赖java11
    安装java11
        dnf install java-11-openjdk java-11-openjdk-devel
    切换版本
        alternatives --config java
    检查JAVA_HOME是否指定非11版本（可不设置，设置后不能是非11版本）
解压安装包
    tar -xvf apache-cassandra-4.1.7-bin.tar.gz
移动到指定目录
    mv /server/apache-cassandra-4.1.7 /server/tool/apache-cassandra
启动cassandra
    /server/tool/apache-cassandra/bin/cassandra -f -R
    报错处理（大多数JVM错误需对jvm-server.options或jvm11-server.options做修改）
        错误：Unrecognized VM option 'UseBiasedLocking'
            vi /server/tool/apache-cassandra/conf/jvm-server.options
                注释掉-XX:-UseBiasedLocking
        错误：Unrecognized VM option 'UseConcMarkSweepGC'
            vi /server/tool/apache-cassandra/conf/jvm11-server.options
                注释掉-XX:+UseConcMarkSweepGC
            # -XX:+UseConcMarkSweepGC
            # -XX:+CMSParallelRemarkEnabled
            -XX:SurvivorRatio=8
            -XX:MaxTenuringThreshold=1
            # -XX:CMSInitiatingOccupancyFraction=75
            # -XX:+UseCMSInitiatingOccupancyOnly
            # -XX:CMSWaitDuration=10000
            # -XX:+CMSParallelInitialMarkEnabled
            # -XX:+CMSEdenChunksRecordAlways
            ## some JVMs will fill up their heap when accessed via JMX, see CASSANDRA-6541
            # -XX:+CMSClassUnloadingEnabled
登录数据库
    /server/tool/apache-cassandra/bin/cqlsh
