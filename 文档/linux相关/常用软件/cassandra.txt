依赖java11
    安装java11
        dnf install java-11-openjdk java-11-openjdk-devel
    切换版本
        alternatives --config java
    检查JAVA_HOME是否指定非11版本（可不设置，设置后不能是非11版本）
解压安装包
    tar -xvf apache-cassandra-4.1.7-bin.tar.gz
移动到指定目录
    mv /server/apache-cassandra-4.1.7 /server/tool/apache-cassandra-4.1.7
启动cassandra
    /server/tool/apache-cassandra-4.1.7/bin/cassandra -f -R
    报错处理（大多数JVM错误只需对jvm-server.options或jvm11-server.options做修改，注释到对应的配置）
        错误：Unrecognized VM option 'UseBiasedLocking'
            vi /server/tool/apache-cassandra-4.1.7/conf/jvm-server.options
                注释掉-XX:-UseBiasedLocking
登录数据库
    /server/tool/apache-cassandra-4.1.7/bin/cqlsh
编辑自启动文件
    vi /etc/systemd/system/apache-cassandra-4.1.7.service
[Unit]
Description=apache-cassandra-4.1.7
After=network.target
[Service]
Type=forking
ExecStart=/server/tool/apache-cassandra-4.1.7/bin/cassandra -f -R
ExecStop=/server/tool/apache-cassandra-4.1.7/bin/nodetool drain
PrivateTmp=true
[Install]
WantedBy=multi-user.target

设置自启动
    systemctl enable apache-cassandra-4.1.7
